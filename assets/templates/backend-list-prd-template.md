## 3. [模块名称]需求说明

### 3.1 筛选查询区

> 描述筛选条件及查询逻辑

| 筛选项 | 组件类型 | 可选值 / 范围 | 默认值 | 说明 |

- 搜索触发方式：点击搜索按钮
- 筛选条件组合规则：AND

### 3.2 列表展示区

> 描述列表字段展示规则

| 字段名称 | 展示格式 | 说明 |
|---|---|---|
| 设备名称 | 文本 | 超过20字符省略显示，hover展示完整名称 |
| SN码 | 文本 | 唯一设备标识，支持复制 |
| 通信方式 | 标签 | 蓝牙（蓝色）/ 4G（绿色）|
| 在线状态 | 状态标签 | 在线（绿色）/ 离线（灰色）/ 故障（红色）|
| 最后上报时间 | YYYY-MM-DD HH:mm:ss | 设备最后一次上报数据的时间 |
| 固件版本 | 文本 | 当前设备固件版本号 |

#### 3.2.1 字段详细说明

##### 在线状态

- **字段说明**：基于设备心跳包判断在线状态
- **判断规则**：距最后心跳时间 ≤5分钟=在线；>5分钟且≤24小时=离线；设备上报故障码=故障
- **展示格式**：彩色状态标签
- **空值展示**：灰色"未知"标签

#### 3.2.2 分页规则

- 每页条数：默认20，可选20/50/100
- 分页方式：页码翻页

### 3.3 操作按钮

| 按钮名称 | 按钮位置 | 点击逻辑 | 权限要求 |
|---|---|---|---|
| 新增设备 | 列表右上角 | 跳转至设备新增页 | 需要编辑权限 |
| 编辑 | 行内操作列 | 跳转至设备编辑页，携带设备ID | 需要编辑权限 |
| 指令下发 | 行内操作列 | 弹出指令下发弹窗 | 需要编辑权限 |
| 删除 | 行内操作列 | 弹出二次确认，确认后删除 | 需要编辑权限 |
| 导出 | 列表右上角 | 导出当前筛选结果为Excel | 需要导出权限 |

#### 3.3.1 指令下发按钮

- **按钮样式**：次要按钮样式
- **显示条件**：设备在线状态为"在线"时可点击，离线时置灰
- **点击逻辑**：
  1. 校验设备在线状态
  2. 在线：弹出指令下发弹窗，选择DP功能点及目标值
  3. 确认下发：通过4G/蓝牙通道推送指令至设备
  4. 下发成功：提示"指令已下发，等待设备响应"
  5. 下发失败：提示具体失败原因（超时/设备离线/协议错误）

#### 3.3.2 删除按钮

- **按钮样式**：危险按钮样式
- **点击逻辑**：
  1. 弹出二次确认："确定删除该设备？删除后设备数据将一并清除且不可恢复。"
  2. 确认：调用删除接口，同步清除设备绑定关系及历史数据
  3. 删除成功：提示成功，刷新列表
  4. 删除失败：显示失败原因

- **编辑限制**：编辑模式下SN码字段只读，不可修改
- **空值展示**：Placeholder："请输入设备SN码"

##### 设备型号

- **字段说明**：选择设备型号后，系统自动关联该型号对应的DP功能点配置
- **联动规则**：型号变更后，DP功能点配置区域同步刷新

| 通信方式 | 选择"4G" | APN/服务器地址配置项 | 显示 |
| 设备型号 | 值变更 | DP功能点列表 | 刷新关联配置 |
